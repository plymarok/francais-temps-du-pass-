📖 Le cours — Rôles des temps

🌸 Imparfait : décor, habitude, action de fond 💚 Passé simple : actions principales, successives 🟧 Plus-que-parfait : « déjà avant » une autre action passée

Imparfait 🌸

  • Description : La ville brillait.
  • Habitude : Il lisait chaque soir.
  • Action de fond : Je marchais quand la pluie commença.

Passé simple 💚

  • Action ponctuelle : Il entra.
  • Successions : Il prit son sac puis sortit.

Plus-que-parfait 🟧

  • Antériorité : action achevée avant un autre passé.
  • Formation : avoir/être à l’imparfait + participe : j’avais terminé, elle était partie.
  • Indices : déjà, avant, lorsque, après que…

Ex. : Quand j’arrivai (💚), elle avait déjà mangé (🟧) → le repas est antérieur à l’arrivée.
